# No Rocket Usage

Rocket flying is annoying! (and unfair, too.)

## Highlights

It prevents firework usage by breaking the equipped elytra upon usage, simple as that.

## Installation

### Datapack

- Make a new world, click on **Datapacks** and put the `.zip` file inside of the `datapacks` folder.
- For existing worlds, put the datapack inside of the `datapacks` folder of that world.

### Mod

- Move the `.jar` file into the `mods` folder.
- Alternatively, you can go to the Mods page and click on **Add File** when using MultiMC or Prism Launcher.
